# Transcription Automation

YouTube video transcription workflow using UniScribe API.

## Structure

- `research/` - Transcriptions for tutorials/research
- `podcast/` - Transcriptions for podcasts (Ultimate Football Heroes, etc.)
- `scripts/` - Python scripts for automation

## Setup

```bash
# Store API key
cp /root/.credentials/uniscribe.json .credentials/uniscribe.json
```

## Usage

Send a YouTube URL → Script transcribes → I ask what to do next.
